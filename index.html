    ABOUT ME

    Hi, my name is Kizito and I graduated from Texas Southern University with a Bachelor's Degree in Computer Engineering. While in school, there were two courses that caught my interest, C++ and Java. Since graduating, I’ve been teaching myself how to program and even worked at Revature as a Software Engineer. At Revature, my duties were to provide maintenance to client’s websites by effectively solving bugs, refactoring some of the code for better run-time efficiency, and offer clients suggestions that would help them attract more audiences for their business. Most recently, I graduated from an immersive Software Engineering program at Flatiron School, where we focused on building projects using Ruby on Rails, JavaScript, and React - my favorite being ZapChat, an app that allows the user to post a chat, comment on it and like it. You can see it in my Project section. There is one other thing I’m interested in and that’s playing video games. If you know of any cool video games out there let me know about it.
